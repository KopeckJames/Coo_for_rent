## üìù Content Writing Tips

### SEO-Friendly Content Structure
1. **Title**: Include primary keyword (60 characters max)
2. **Meta Description**: Compelling, includes keyword (155 characters)
3. **H1**: One per page, includes primary keyword
4. **H2/H3**: Include secondary keywords naturally
5. **First Paragraph**: Include primary keyword in first 100 words
6. **Body**: Use keywords naturally (1-2% density)
7. **Images**: Alt text with keywords
8. **Internal Links**: Link to related pages
9. **CTA**: Clear call-to-action
10. **Length**: 800-2000 words for service pages, 1500+ for blog posts

### Writing for Humans AND Google
- Write naturally, don't stuff keywords
- Answer user questions directly
- Use short paragraphs (2-3 sentences)
- Include bullet points and lists
- Add examples and case studies
- Use conversational tone
- Include numbers and data
- Make it scannable
